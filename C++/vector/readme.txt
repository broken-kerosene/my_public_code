Реализуйте свой класс Vector, аналогичный std::vector<int>. Эта задача простая: требуется, чтобы ваш класс работал только с числами типа int (то есть, не был бы шаблонным) и предоставлял функции push_back, pop_back, size, operator[], а также конструктор без параметров, конструктор копирования, оператор присваивания и деструктор. Ваш класс должен непосредственно работать с динамической памятью: пользоваться аналогичными классами стандартной библиотеки запрещено.
